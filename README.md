# How To

1. Run the app
    ```shell
    npm install
    node app.js
    ```
1. See results in app log:
    ```
    Server started on port 5000
    ```
1. Open http://127.0.0.1:5000/
1. See results in app log:
    ```
    Server started on port 5000
    A new socket has joined: IqEjjc0dBHYSHqpMAAAB
    ```
